Salah Kaprah yang Umum Mengenai World Wide Web

August 3, 2013

Ini adalah bagian dari eBook Memahami Dunia Pemrograman Web untuk Pemula. Beberapa tulisan masih membutuhkan untuk disunting kembali. Tags untuk eBook ini adalah buku-web sehingga bisa dicari di blog ini.

WWW

World Wide Web ( disini saya singkat menjadi Web ) adalah sebuah sistem yang menghubungkan berbagai bentuk dokumen dalam bentuk hypertext menggunakan medium internet.

Jadi, web bukanlah Internet secara keseluruhan. Web adalah sebuah bagian dari Internet.

Kemudian, apa itu hypertext?

Hypertext adalah teks yang ditampilkan di layar komputer, layar handphone, atau layar perangkat elektronik lainnya.

Caranya dihubungkan bagaimana?

Dengan menggunakan hyperlink yang merupakan entitas referensi.

Bagaimana ditampilkannya?

Web ditampilkan menggunakan browser. Beberapa produk browser yang paling umum digunakan adalah Firefox dari Mozilla Foundation dan Google Chrome.

Bagaimana Cara Membuka sebuah Web?

Internet memiliki web yang sangat banyak. Ada jutaan, bahkan milyaran web di Internet. Untuk dapat mengakses web, dibutuhkan URL. Sebagai contohnya http://www.google.com

Protokol

Dalam contoh ini, URL yang digunakan adalah untuk membuka Web Search Engine dari Google.

URL dibagi menjadi 3 bagian dengan format protocol://server/path.

Ketika halaman web dibuka dengan laptop, komputer atau handphone anda, di ujung lainnya juga terdapat sebuah komputer yang mengetahui bahwa anda ingin membuka Search Engine Google. Ini yang dinamakan dengan Web Request.

Request anda akan diolah oleh sebuah aplikasi yang dinamakan webserver. Ketika request anda mencapai webserver ini, maka webserver akan memberikan balikan berupa halaman Google. Ini yang dinamakan Web Response.

Untuk lebih mudahnya, web bekerja dengan konsep request dan response. Request adalah permintaan yang dikirimkan oleh client, sedangkan Response adalah balikan dari server terhadap request yang diminta.

Domain dan IP Address

Ketika anda mengetikkan URL Google diatas, sebenarnya request anda akan ditujukan terhadap sebuah mesin komputer tertentu yang memiliki alamat IP.

Sebagai analoginya, anggaplah anda memiliki teman bernama Johan. Johan, memiliki alamat di Vila Pamulang Mas, Blok 7/21. Alamat ini bisa dianalogikan sebagai IP address, sedangkan anda, karena lebih mudah mengatakannya, hanya tinggal menyebut "Rumah Johan".

Begitu pula di dunia web. Web dari Google memiliki IP address tertentu. Sebagai percobaan, Anda dapat membuka browser dan mengetikkan ini http://74.125.235.36. Hasilnya akan sama bila anda mengetikkan http://www.google.com.

Penamaan Google.com inilah yang disebut dengan Domain Address. Ketika request anda diterima, maka akan ada sebuah service, yang dinamakan DNS dimana akan mengubah dari Google.com menjadi IP address 10.x.x.x dan akan meneruskan kepada web terkait.

Ini adalah gambaran dasar dan umum mengenai World Wide Web. Konsep fundamental ini sangat dibutuhkan untuk memahami bahasan di bab berikut.
