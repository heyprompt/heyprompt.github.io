Adding Google Analytics for Jekyll

June 22, 2014

I have to admit that Jekyll is a really great platform. No wonder since it has been developed by the founder of Github, Tom Preston-Werner.

However, it lacks of few features that unfortunately have become the mandatory requirements for web. Google Analytics and SEO are two highlights among the other features.

Jekyll Bootstrap?

Most people get confused when they use Jekyll Bootstrap which you can automatically insert the Google Analytics code, but for other themes, you can't because some not include the Google Analytics feature in it. In this post, I'll try to explain how to add the Google Analytics in most Jekyll themes and not Jekyll Bootstrap.

The steps are fairly straight forward:
- Get your Analytics Code - Identify which base HTML template that includes the head script - Paste the Analytics Code

Let's Do It

Analytics Code

You can get your Analytics code by logging into Google Analytics and get the code under the Admin tab and find "Tracking Info".

Under the option, you will see "Tracking Code" and the code that you will use.

Head Base HTML

This part is quite tricky for you that don't understand Jekyll layout theming. But, you can identify which file by looking under the:

_includes

This folder holds all layouting in Jekyll. You can open one by one and search for any file that contains this directive:

<head>
....
</head>

Paste The Analytics Code

Now you can paste the code above the </head>. At the end, it would be exactly like this:

<script>
(function(i,s,o,g,r,a,m) {i['GoogleAnalyticsObject']=r;i[r]=i[r]||function(){
(i[r].q=i[r].q||[]).push(arguments)},i[r].l=1*new Date();a=s.createElement(o), m=s.getElementsByTagName(o) [0];a.async=1;a.src=g;m.parentNode.insertBefore(a,m) })(window,document,'script','//www.google-analytics.com/analytics.js','ga');
ga('create', 'UA-xxxxxx-xx', 'xxxxx.com');
ga('send', 'pageview');
</script>
</head>

That's it. Now you have your analytics code implemented and it will start rolling in few minutes.

Happy Blogging!
